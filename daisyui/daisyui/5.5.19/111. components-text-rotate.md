# components-text-rotate

> Text Rotate can show up to 6 lines of text, one at a time, with a an infinite loop animation. Duration is 10 seconds by default. The animation will pause on hover.

| Class name  | Type      |                   |
| ----------- | --------- | ----------------- |
| text-rotate | Component | Wrapper for texts |

Text Rotate can show up to 6 lines of text, one at a time, with a an infinite loop animation. Duration is 10 seconds by default. The animation will pause on hover.

[](#text-rotate)

### Text Rotate

Rotates through 3 words, in 10 seconds

ONE TWO THREE

```
<span class="$$text-rotate">
  <span>
    <span>ONE</span>
    <span>TWO</span>
    <span>THREE</span>
  </span>
</span>
```

```
<span class="$$text-rotate">
  <span>
    <span>ONE</span>
    <span>TWO</span>
    <span>THREE</span>
  </span>
</span>
```
