## Summary

[Reference Documentation ↗](https://api.reactrouter.com/v7/functions/react_router.Meta.html)

Renders all the [`<meta>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta) tags created by the route module's [`meta`](about:blank/start/framework/route-module#meta) export. You should render it inside the [`<head>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/head) of your document.

```
import { Meta } from "react-router";

export default function Root() {
  return (
    <html>
      <head>
        <Meta />
      </head>
    </html>
  );
}
```
