## Summary

[Reference Documentation ↗](https://api.reactrouter.com/v7/functions/react_router.useLoaderData.html)

Returns the data from the closest route [`loader`](about:blank/start/framework/route-module#loader) or [`clientLoader`](about:blank/start/framework/route-module#clientloader).

```
import { useLoaderData } from "react-router";

export async function loader() {
  return await fakeDb.invoices.findAll();
}

export default function Invoices() {
  let invoices = useLoaderData<typeof loader>();
  // ...
}
```

## Signature

```
function useLoaderData<T = any>(): SerializeFrom<T>
```

## Returns

The data returned from the route's [`loader`](about:blank/start/framework/route-module#loader) or [`clientLoader`](about:blank/start/framework/route-module#clientloader) function
