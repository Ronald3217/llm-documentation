## Summary

[Reference Documentation ↗](https://api.reactrouter.com/v7/functions/react_router.useRouteError.html)

Accesses the error thrown during an [`action`](about:blank/start/framework/route-module#action), [`loader`](about:blank/start/framework/route-module#loader), or component render to be used in a route module [`ErrorBoundary`](about:blank/start/framework/route-module#errorboundary).

```
export function ErrorBoundary() {
  const error = useRouteError();
  return <div>{error.message}</div>;
}
```

## Signature

```
function useRouteError(): unknown
```

## Returns

The error that was thrown during route [loading](about:blank/start/framework/route-module#loader), [`action`](about:blank/start/framework/route-module#action) execution, or rendering
