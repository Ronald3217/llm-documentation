# elements-delete

> The delete element is a stand-alone element that can be used in different contexts.

| CSS Variable                         | Value              |
| ------------------------------------ | ------------------ |
| var(--bulma-delete-dimensions)       | 1.25rem            |
| var(--bulma-delete-background-l)     | 0%                 |
| var(--bulma-delete-background-alpha) | 0.5                |
| var(--bulma-delete-color)            | var(--bulma-white) |

The `delete` element is a stand-alone element that can be used in different contexts.

On its own, it's a simple circle with a cross:

Example

HTML

```
<button class="delete"></button>
```

### Sizes [#](#sizes)

It comes in **4 sizes**:

Example

HTML

```
<button class="delete is-small"></button>
<button class="delete"></button>
<button class="delete is-medium"></button>
<button class="delete is-large"></button>
```

### Combinations [#](#combinations)
